About My Java Journey

During my second semester at VIT Chennai, I was introduced to Java programming along with Object Oriented Programming. I found this versatile language to be rich and powerful. My learning journey involved a combination of resources, including YouTube tutorials, online articles, and courses, complemented by classroom instruction and hands on practice sessions at college laboratory as well as self practice sessions. This comprehensive approach helped me gain a solid understanding of Java.

For my end-semester project, I individually developed a Supermarket Inventory Management System using Java, JDBC, and MySQL. The system efficiently handled both administrative tasks and customer-side operations related to shopping and inventory management. I was awarded the top grade in my class for this project.

Currently, Java remains my primary language, and I am actively working to further strengthen my grasp of it.
